What this app does

📅

Family Calendar

Monthly view with multi-event days, date-range events, member color coding, and 12-hour time format. Medical, school, vehicle, home & custom event types.

🚗

Auto Maintenance

Full service history for every vehicle. Tracks odometer (km), cost with HST calculator, duration, technician, and file attachments — photos, PDFs, invoices, or any document.

🏡

Home Schedules

HVAC filters, appliances, plumbing, electrical — every home maintenance task logged with cost, service provider, and file attachments — photos, PDFs, invoices, warranties, or any safe document.

📊

Reports & Summaries

Filterable spend reports across Auto, Home, and Calendar. Total cost tracking with HST breakdowns. Printable layouts.

🔒

Private & Secure

Google Sign-In required. Role-based access (Admin / Contributor / Viewer). Unknown accounts are blocked outright — household members only.

📱

Mobile-First PWA

Fully responsive design optimized for Android Chrome. Installable as a Progressive Web App with offline support and a native app feel.

🤖

Native Android APK

A real Android app (Capacitor v6) built and published automatically via GitHub Actions on every update. No app store needed.

⚙️

Full Admin Control

Manage vehicles, family members, categories, branding, and user access — all from within the app. All config stored in real-time in Firebase.

🛠️ Tech Stack

Firebase Firestore Firebase Auth Firebase Storage GitHub Pages GitHub Actions Capacitor v6 PWA Vanilla JS HTML5 / CSS3 Google Sign-In GoDaddy DNS Let's Encrypt HTTPS
